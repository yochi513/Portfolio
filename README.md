## Tystar(unity)

### 概要
3Dタイピングゲーム

### 期間や感想
- 2回目のチーム制作になります。
- こちらも8カ月かけて制作しました。
- 分野の偏りがありプログラマー1人デザイン4人企画3人で制作しました。
- ほとんど自分がコードを担当しましたが、エフェクトやBGM・SEのコードは担当していません。

- 反省点は分野を偏らせすぎたこと。タスク管理が疎かになり現状誰がどこまで進んでいるのかがわからなくなってしまった時があった。
- 会議の回数が少なく授業で集まれた時に会議をしたり、そもそも他メンバーの出席率がバラバラであったこともありました。

- 改善策として会議は週2とし、タスク管理は作業をする前に一度アプリを開いてから作業に取り掛かるようにしました。
出席率がバラバラであるときは必ず進捗報告を送ってもらうことを徹底しました。
